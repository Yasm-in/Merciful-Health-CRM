# Merciful Health CRM

A lightweight healthcare CRM for small clinics to manage patients, status updates, and follow-up actions.

## Features
- Add and track patient records
- Set follow-up dates
- Manage treatment status workflow
- Store notes per patient

## Tech Stack
- Python
- Flask
- SQLite
- HTML/CSS/JavaScript

## Run Locally
```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

Open `http://127.0.0.1:5000`.

## Quick API Verification
```bash
python test_api.py
```

Expected output: `All Merciful Health CRM API checks passed.`
